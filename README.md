# My dotfiles and configuration scripts

## Installation
> [!WARNING]
> Will override the configuration directories with symlinks

### HTTPS
`curl -sL https://raw.githubusercontent.com/ariannelafraise/october-dotfiles/refs/heads/main/setup-https.sh | bash`

### SSH
`curl -sL https://raw.githubusercontent.com/ariannelafraise/october-dotfiles/refs/heads/main/setup-ssh.sh | bash`
